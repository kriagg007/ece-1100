# Krishiv Aggarwal — ECE ePortfolio

A static, single-page ePortfolio tailored for the ECE 1100 assignment. Includes all required sections and
content customized to Krishiv's background.

## Quick Publish (GitHub Pages)
1. Create a new **public** GitHub repo.
2. Upload all files from this folder (keep structure).
3. In repo **Settings → Pages**, set Source to **Deploy from a branch**, Branch **main / root** → **Save**.
4. Your site will appear at: `https://<username>.github.io/<repo>/`

## Netlify Drop (1‑click deploy)
1. Go to https://app.netlify.com/drop
2. Drag the **unzipped** folder to deploy and get a live URL instantly.
