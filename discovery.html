Discovery Project: 5V → 15V Boost Converter

A hands-on power electronics project exploring switching converters, PWM control, inductor behavior, diode selection, and real-world vs. simulated performance.

Idea Progress Skills Gained

Starting Idea / Pitch

Project Motivation

My original idea was to build a hardware project that would help me understand switching behavior, inductors, MOSFET control, and the basics of power conversion. A boost converter is a classic and extremely useful topology — found in battery-powered devices, LED drivers, and RF circuits.

The goal: take a 5V input and step it up to ~15V using an inductor, MOSFET, diode, and PWM control.

Initial Plan

  • Schematic design in QSPICE
  • Simulate current/voltage behavior at different duty cycles
  • Breadboard prototype using AO4262E MOSFET + ES3A diode
  • Measure ripple, startup behavior, and load response

Breadboard prototype of the 5V → 15V converter.

Project Progress

Week 1Learned boost converter topology and duty-cycle equation: Vout = Vin / (1 - D)
Week 2Built QSPICE schematic and simulated switching behavior
Week 3Selected components: 22µH inductor, ES3A diode, AO4262E MOSFET
Week 4Breadboarded full converter; verified diode conduction + inductor charging
Week 5Measured output ripple and validated simulated overshoot

Schematic & Simulation

QSPICE schematic using AO4262E MOSFET and ES3A diode.

Simulation showing startup overshoot and settling behavior.

The simulation predicted a voltage overshoot to ~22V before settling near 15V, which matched real hardware behavior closely. This validated the inductor value and switching frequency selection (~200 kHz).

Successes & Failures

Successes

  • Simulation matched real-world waveform shape
  • Achieved ~15V output from 5V input
  • Inductor + MOSFET switching worked reliably
  • Learned reading QSPICE waveforms (overshoot, ripple)

Failures / Challenges

  • Large initial overshoot (~22V) before settling
  • Output ripple higher than expected
  • Inconsistent behavior from breadboard parasitics
  • ES3A diode forward drop caused efficiency loss

Final Thoughts & Reflection

This project gave me my first real hands-on experience with switching converters, and it strengthened my interest in power electronics. Seeing the simulation match the real circuit was extremely rewarding. I plan to continue this work by designing a proper PCB version of this boost converter and characterizing efficiency more deeply.

Overall, this project validated my long-term interest in the Power & Electronic Devices thread.
