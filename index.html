Technical Projects

Discovery Project — to be finalized for Showcase

LED Progress Bar — Visualizing Task Completion

Role: Designer & Builder • Tools: VHDL, FPGA I/O, Breadboard • Status: In progress

This project implements a visual “progress bar” using LEDs to represent a percentage from user inputs. It explores digital logic design, state-machine planning, and hardware interfacing. The current prototype integrates simple input parsing and PWM-based brightness control for smoother gradients. For the Showcase, I will add improved input handling, packaging, and documentation.

Objectives

  • Translate numeric progress into LED states and brightness.
  • Demonstrate reliable hardware timing and debouncing.
  • Create a clear demo video and schematic for documentation.

Results (So Far)

  • Functional prototype with adjustable thresholds.
  • Measured stable timing and uniform LED response.
  • Next: enclosure, refined UI, and expanded testing.
